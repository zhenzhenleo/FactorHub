# FactorHub

**FactorHub** は中国のA株市場向けに設計されたオープンソースの現代的な定量的ファクター分析プラットフォームです。

> FactorHub = Factor（ファクター） + Hub（中心）

ファクター管理、分析、マイニング、ポートフォリオ最適化、戦略バックテストを統合したフルスタックの定量的投資研究システム。

## コア機能

![1773500695533](image/README_ZH/1773500695533.png)

### 1. ファクター管理
- ✅ **カスタムファクター定義** - 通达信（マイランゲージ）構文とTALib関数をサポート
- ✅ **式の検証** - リアルタイムの構文チェックと論理検証
- ✅ **バージョン管理** - ファクターの変更履歴とバージョンロールバック
- ✅ **プレビルトファクターライブラリ** - MA、RSI、MACD、ボリンジャーバンドなどの一般的なテクニカルファクターを内蔵

![1773500664940](image/README_ZH/1773500664940.png)

### 2. ファクター分析
- ✅ **IC/IR分析** - 情報係数と情報比率の計算（1日、5日、10日の予測期間に対応）
- ✅ **ファクター暴露分析** - 株式のファクターへの暴露分布の分析
- ✅ **ファクター有効性検査** - ファクターの予測能力の多次元評価
- ✅ **ファクター属性分析** - ファクターの収益への寄与の分解
- ✅ **動的モニタリング** - 時系列次元におけるファクターのパフォーマンス追跡

![1773500638704](image/README_ZH/1773500638704.png)

### 3. ファクターマイニング
- ✅ **遺伝的アルゴリズムマイニング** - DEAPに基づく進化的アルゴリズムによる自動ファクター探索
- ✅ **多目的最適化** - IC、IR、単調性などの複数の目標を同時に最適化
- ✅ **ファクター生成** - 基本的な演算子、関数呼び出し、時間窓操作をサポート
- ✅ **並列計算** - 個体群の並列評価をサポートし、マイニングプロセスを高速化

![1773500591756](image/README_ZH/1773500591756.png)

### 4. ポートフォリオ分析
- ✅ **マルチファクターポートフォリオ** - 等加重、時価総額加重、IC_IR最大化などのコンビネーション方法をサポート
- ✅ **リスクモデリング** - ファクターニュートラル化処理
- ✅ **最適化構成** - 履歴パフォーマンスに基づくファクターの重みの最適化
- ✅ **ポートフォリオパフォーマンス** - 年間収益、シャープレシオ、最大ドローダウンなどの指標

![1773500498447](image/README_ZH/1773500498447.png)

### 5. 戦略バックテスト
- ✅ **シングルファクターバックテスト** - ファクターの分位数に基づく株式選択バックテスト
- ✅ **マルチファクター戦略** - 複合ファクター信号の生成
- ✅ **戦略比較** - マルチストラテジーの並列バックテストと比較分析
- ✅ **パフォーマンス指標** - 収益、リスク、ターンオーバーなどの完全な指標体系
- ✅ **可視化チャート** - 権益曲線、ドローダウン、ファクターのパフォーマンスなどのチャート

![1773500440529](image/README_ZH/1773500440529.png)

## クイックスタート

### 前提条件

- **Python 3.11+**
- **Node.js 18+**
- **pnpm** (パッケージマネージャー)
- **TA-Lib** (テクニカル分析ライブラリ)

### インストール

```bash
# Install pnpm
npm install -g pnpm

# Install Python dependencies (using uv)
uv sync

# Install frontend dependencies
cd frontend/react-antd
pnpm install
```

### ワンクリックスタートアップ

```bash
python start_all.py
```

このスクリプトは自動的に以下を実行します:
1. 環境の前提条件のチェック
2. 必要に応じて依存関係のインストール
3. バックエンドサービスの起動（http://localhost:8000）
4. フロントエンド開発サーバーの起動（http://localhost:5173）
5. ブラウザの自動オープン

### 手動スタートアップ

#### バックエンド

```bash
uv run python start_api.py
# API available at http://localhost:8000
# API docs at http://localhost:8000/docs
```

#### フロントエンド

```bash
cd frontend/react-antd
pnpm dev
# Frontend available at http://localhost:5173
```
